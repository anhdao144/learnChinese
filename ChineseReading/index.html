Từ Vựng Đã Sử Dụng

Hướng Dẫn Sử Dụng

Trang web này sử dụng Openrouter API để tạo bài luyện đọc tiếng Trung dựa trên từ vựng HSK mà bạn cung cấp.

1 Nhập từ vựng

Nhập từ vựng tiếng Trung bạn muốn luyện tập. Bạn có thể nhập kèm pinyin và nghĩa tiếng Việt.

2 Chọn cấp độ HSK

Chọn cấp độ HSK phù hợp để bài đọc được điều chỉnh độ khó tương ứng.

3 Nhập API Key (tùy chọn)

Nhập API Key từ Openrouter hoặc sử dụng chế độ demo để thử nghiệm.

4 Tạo bài luyện đọc

Nhấn nút "Tạo Bài Luyện Đọc" và chờ hệ thống tạo nội dung phù hợp với từ vựng của bạn.
